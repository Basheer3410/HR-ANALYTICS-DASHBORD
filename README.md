# HR-ANALYTICS-DASHBORD

This project presents an interactive HR Analytics Dashboard built entirely in Microsoft Excel.
It analyzes employee data across departments, age groups, gender distribution, and hiring sources.
Pivot tables, slicers, and dynamic charts are used for data visualization and insights.
The dashboard enables quick workforce analysis and better HR decision-making.
This project demonstrates strong Excel skills in data analysis and dashboard creation.
